# Console App

Operator UI boundary for the public Flion Agentic Super App demo.

Planned responsibilities:

- show chat, command palette, plugin/skill registry, role agents, automations, and release readiness
- expose the Data Cloud Agents vertical slice as runnable evidence
- keep private runtime state out of the public console
- keep manual operator decisions separate from agent-generated evidence

This folder starts as an architectural boundary. Implementation should arrive after contracts and services are stable.

Current implementation:

- `index.html`: static operator console shell
- `styles.css`: responsive data-dense visual system
- `app.js`: renders generated super-app and demo evidence
- `demo-data.json`: generated by `npm run console:build`

Run locally:

```bash
npm run console:serve
```
